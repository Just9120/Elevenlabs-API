Status vocabulary:

- `source-done/merged` ‚Äî repository source/docs merged to `main`.
- `CI-verified` ‚Äî relevant GitHub Actions checks passed.
- `deployed` ‚Äî factual web/API component deployment evidence exists for a specific commit and component.
- `migration-applied` ‚Äî factual operator evidence shows the production database reached a specific Alembic revision.
- `worker-running` ‚Äî factual operator evidence shows the intended worker process is running.
- `production-live` ‚Äî factual end-to-end runtime/operator evidence exists for successful production processing; merge, CI, deployment, migration, API health, or idle worker evidence alone does not prove this state.
- `manual-ops-required` ‚Äî cannot be completed by a coding agent alone.
- `deferred` ‚Äî intentionally outside current PR scope.
- `blocked` ‚Äî requires decision, access, migration, or runtime work.
